# Chat Handoff Checklist

## Required

1. Run `npm run handoff`.
2. Confirm whether validation passed.
3. Point the next chat to `AGENTS.md`, `docs/agents/README.md`, and `docs/agents/chat-handoff.md`.

## Add Manual Notes When Needed

- Outstanding bug or failing test
- Known data dependency or missing workbook file
- Follow-up work that was intentionally deferred
